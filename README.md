# arduino-snake
A Snake Game with Arduino 

# Components Required:
- 1 x Elegoo Mega 2560 R3
- 1 x MAX7219 LED Dot Matrix Module
- 1 x Joystick module
- 10 x F-M wires (Female to Male DuPont wires)

# Instructions
### Connect Joystick
 - `SW` to `D2 PWM`
 - `VRx` to `Analog 0`
 - `VRy` to `Analog 1`
 - `+5V` to `5V`
 - `GND` to `GND`
 
### Connect LED Dot Matrix
- VCC to `5V`
- GND to `GND`
- DIN to `D2 PWM 12`
- CS to `D2 PWM 11`
- CLK to `D2 PWM 10`
